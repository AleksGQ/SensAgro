# SensAgro
**SensAgro** es un sub-proyecto que hace parte del proyecto "Desarrollo de una plataforma de agricultura inteligente y autónoma energéticamente para el monitoreo continuo de variables relevantes orientada a mejorar la productividad y mitigar el impacto ambiental en cultivos hortofrutícolas de Antioquia, Quindío." denominado **PAI** (Plataforma de Agricultura Inteligente). El objetivo principal de **SensAgro** es la creación de un _testbed_ para la evaluación de alternativas de conectividad en soluciones de IoT/AIoT, en este caso orientadas a apoyar los procesos en el sector agrícola.
La idea es probar tecnologías como LoRA/LoRAWAN, WiFi, WiFi HaLow, Threat, Bluetooth, ZigBee, celulares 2/3/4/5G y relacionadas, y comparar soluciones particulares en términos de consumo de energía, cobertura, capacidad, latencia, entre otras. Adicionalmente, se busca seguir de cerca la evolución del proceso de estandarización de 6G en lo relativo a IoT y tecnologías de transición como 5G RedCap, para determinar el impacto que tendrán estas aplicaciones en las redes futuras, los retos asociados y las posibles soluciones.
